# **🏫 단어 찾기 (Word Search Classroom)**

교실과 가정에서 쉽고 재미있게 어휘를 학습할 수 있도록 설계된 **웹 기반 단어 찾기 퍼즐 게임**입니다. 사용자가 직접 수동으로 편집하거나 엑셀(Excel) 파일을 업로드하여 단기간에 맞춤형 학습지를 무제한으로 생성할 수 있습니다.

## **🌟 주요 특징 (Key Features)**

* **동적 단어 배치 알고리즘**: 게임 번호(난수 시드)를 기반으로 매번 새로운 단어 배치판을 생성합니다.  
* **자음(초성) 힌트 토글**: 뜻풀이 우측의 전구 아이콘을 누르면 초성 힌트가 나타나며, 다시 누르면 가릴 수 있어 학습 편의성을 높였습니다.  
* **대용량 엑셀 등록 및 무한 자동 분할**:  
  * 엑셀 파일(.xlsx, .xls, .csv) 하나로 손쉽게 맞춤 문제를 등록합니다.  
  * 단어 글자 수(2\~8자) 유효성 검사를 제공하며, **1글자 단어는 퍼즐 규칙상 자동으로 제외**하고 안내합니다.  
  * **단어 수는 40개로 자동 제한**되며, 40개가 넘는 파일을 올리면 40개 단위로 쪼개어 **새 페이지를 자동으로 열어** 연쇄적으로 다음 게임들을 만들어줍니다.

## **🎮 게임 방법 (How to Play)**

1. **단어 찾기**: 그리드 판에서 우측 뜻풀이에 해당하는 단어를 찾아 마우스 클릭 또는 터치 상태로 쭉 드래그합니다.  
2. **선택 방향**: 가로, 세로, 대각선 및 반대 방향까지 모두 인정됩니다.  
3. **힌트 활용**: 단어가 너무 어렵다면 뜻풀이 리스트의 💡(전구) 버튼을 클릭해 단어의 **초성**을 엿볼 수 있습니다. 다시 누르면 힌트가 가려집니다.

## **📊 엑셀 파일 기입 룰 (Excel Formatting)**

나만의 단어장을 구성할 때는 아래의 규칙을 준수하여 엑셀 파일을 저장한 뒤 업로드해 주세요.

| A열 (정답 단어) | B열 (설명 1 \- 기본 뜻) | C열 (설명 2 \- 외국어 예문 / 역주) |
| :---- | :---- | :---- |
| **선생님** | 학생을 가르치는 사람 | 先生 |
| **정말** | 참되고 거짓 없는 마음 | 거짓이 아님 |

* **주의 사항**:  
  * 1행에 '정답', '단어' 등 타이틀 행이 있어도 자동으로 감지하여 스킵합니다.  
  * 정답 단어는 공백을 제외하고 **2글자 이상 8글자 이하**여야 합니다.  
  * **1글자 단어**는 퍼즐 판 구성 특성상 무조건 **자동 제외**됩니다.  
  * 단어가 **40개를 초과**하면 40개씩 끊어서 자동으로 새 브라우저 창에서 다음 단어 찾기 게임들이 연이어 열립니다. (팝업 차단 해제 필요)

## **🛠 사용된 기술 및 라이브러리 (Tech Stack)**

* **UI/Styles**: [Tailwind CSS v3](https://tailwindcss.com/)  
* **Icons**: [Lucide Icons](https://lucide.dev/)  
* **Excel Parser**: [SheetJS (xlsx.full.min.js)](https://sheetjs.com/)  
* **Typography**: Noto Sans KR, Noto Sans JP  
* **Random Engine**: Mulberry32 Algorithm (Seeded PRNG)

## **📂 실행 및 배포 방법**

본 프로그램은 단 하나의 완성된 HTML 파일(word\_search\_classroom.html)로 이루어져 있어 별도의 서버 설정 없이 독립적으로 동작합니다.

1. word\_search\_classroom.html 파일을 다운로드합니다.  
2. 파일을 더블 클릭하여 Chrome, Edge, Safari 등 웹 브라우저로 엽니다.  
3. 오프라인 환경에서도 작동하지만, 아이콘 및 스타일링 원활을 위해 최초 로드 시에는 인터넷 연결(CDN 링크 수신)을 권장합니다.
